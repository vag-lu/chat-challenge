API reference

Base URL: http://localhost:3001

GET /channels

Returns a list of all channels.

Channel attributes

Attribute Description Type
id Channel ID string
name Channel name string

Example response body

[
    {
      "id": "273",
      "name": "general"
    },
    {
      "id": "764",
      "name": "random"
    }
  ]

GET /messages?_expand=user

Returns a list of all messages with its associated user.

Message attributes

Attribute Description Type
id Message ID string
timestamp Message timestamp string
channel_id ID of channel message belongs to string
content Message content string
is_unread Whether message has been read boolean
user Associated user object
user.id Associated user ID string
user.username Associated user's username string

Example response body

[
    {
      "id": "704",
      "timestamp": "1:40 PM",
      "channel_id": "273",
      "content": "How much wood could a woodchuck chuck if a woodchuck could chuck wood?",

        "id": "405",
        "username": "vito"
      }

      "id": "140",
